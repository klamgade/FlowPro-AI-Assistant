# 1. Common Client Pain Points

These are the real problems this service should solve.

### Writing Problems
Clients struggle with:
- replying to emails quickly
- sounding professional
- writing follow-ups
- rewriting messy drafts

### Information Overload
Clients struggle with:
- long emails
- long reports
- proposals
- contracts

### Thinking Clarity
Clients struggle with:
- structuring ideas
- evaluating options
- deciding what to do next

### Research Time
Clients struggle with:
- comparing tools
- understanding markets
- learning new topics

# 2. Improvements to the Session

Possible improvements to test during pilots.

### Idea 1 – Start with a real task

Instead of explaining prompts first:

Ask the client:

"What is something you had to write this week?"

Then demonstrate AI on that task.

This immediately creates value.

---

### Idea 2 – Show before/after

Demonstrate:

Messy email → AI rewrite

This makes the impact obvious.

---

### Idea 3 – Focus on 2–3 use cases

Do not overload clients with all five.

Prioritise:

- Email assistant
- Document summary
- Research assistant

The rest can be shown briefly.

# 4. Common Client Objections

Prepare responses.

### "I already use ChatGPT"

Response direction:

"Great — most people use it occasionally, but the value comes from using it systematically in daily work."

---

### "I don't have time to learn AI"

Response direction:

"That's exactly why the session exists — we set it up together so you don't have to figure it out alone."
