# AI Work Assistant Setup – Client Guide

## Overview

This document summarises the AI workflows we set up during your session.

The goal is to help you use AI as a **daily work assistant** to save time and improve clarity in your work.

You do not need to use all workflows at once. Start with one or two that match your daily tasks.

Over time, these workflows can become part of your regular work routine.

---

# How to Think About AI

AI works best when treated as:

- a **writing assistant**
- a **research assistant**
- a **thinking partner**

It helps with:

- structuring ideas
- summarising information
- improving communication
- exploring options

You remain responsible for decisions and final judgement.

---

# Workflow 1 – Email Assistant

### When to Use

Use AI when you need to:

- rewrite messy drafts
- respond to client emails
- write follow-ups
- improve tone or clarity
- shorten long messages

### Example Prompt

Rewrite the following email so that it is clear, professional and concise.

Email:

[Paste your email here]

### Example Use

Instead of spending 10 minutes rewriting an email, paste your draft into AI and ask for a cleaner version.

You can then quickly review and send.

---

# Workflow 2 – Document Summary Assistant

### When to Use

Use this workflow when reviewing:

- reports
- proposals
- long emails
- contracts
- research documents

### Example Prompt

Summarise the following document.

Provide:

1. Short summary  
2. Key points  
3. Actions required  
4. Risks or important notes

Document:

[Paste document here]

### Example Use

If a document is long, ask AI to summarise it before reading in detail.

This helps you quickly understand the main ideas.

### When to Use

Use AI when you want to:

- compare tools
- explore opportunities
- research markets

### Example Prompt

1. Simple explanation  
2. Key insights

Instead of spending hours searching multiple websites, use AI to quickly get an overview and key considerations.

# Best Practices for Using AI

### 1. Give Clear Instructions

Instead of:

Write email

Try:

Rewrite this email to sound more professional and concise.

---

### 2. Provide Context

Explain what the task is and who the message is for.

Example:

Rewrite this email to a client who requested a proposal update.

---

### 3. Review the Output

AI is very helpful but should always be reviewed before sending or using the result.
